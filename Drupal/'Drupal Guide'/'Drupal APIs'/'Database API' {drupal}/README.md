Уровень абстракции базы данных Drupal предоставляет единый API запросов к базе данных, который позволяет делать запросы к различным базам данных. Он построен на PDO (PHP Data Objects) API базы данных и наследует большую часть синтаксиса и семантики.

Помимо предоставления унифицированного API для запросов к базе данных, уровень абстракции базы данных также предоставляет структурированный способ построения сложных запросов, а также защищает базу данных, используя хорошие практики безопасности.
